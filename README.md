# Personal Site

Personal homepage built with Hugo and the PaperMod theme.

## Development

- Run local server:
  - `make dev`
- Build site:
  - `make build`
- Clean generated files:
  - `make clean`

## Docker

- Build image (local baseURL):
  - `make docker-build`
- Run image:
  - `make docker-run`
- Visit:
  - http://localhost:8080/
